# 🚗 Sistema de Veículos em C#

Projeto desenvolvido para praticar conceitos de Programação Orientada a Objetos (POO) utilizando C#.

## 📚 Conceitos aplicados

* Herança
* Polimorfismo
* Encapsulamento
* Listas de objetos

## ⚙️ Funcionalidades

* Cadastro de carros
* Cadastro de motos
* Listagem de veículos
* Cálculo de preço final

## 🛠 Tecnologias

* C#
* .NET

## 📌 Objetivo

Projeto criado com foco em aprendizado e evolução na área de desenvolvimento de sistemas.
